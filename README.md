# Comparison-of-Region-Based-on-Sales
This project uses the Sample Superstore Dataset to compare sales performance between two regions using an interactive dashboard in Tableau. The dashboard displays key metrics such as Total Sales, Average Sales per Order, Number of Customers, and Orders for both selected regions.
Project Summary

The project focuses on comparing sales performance between two regions using the Sample Superstore dataset. A dashboard was created to help management visualize and analyze key sales metrics, enabling informed decisions and recommendations for improvement.

Tools Used

Data Source: Sample Superstore dataset

Data Preparation & Organization: Grouping by Customer Name and Order ID in folders, creating a Location hierarchy using Country

Parameters & Calculations:

Parameters: Primary Region and Secondary Region

Calculated Fields: Sales metrics for selected regions and First Order Date

Visualization: Tableau (or similar BI tool) for creating the dashboard

Metrics Displayed:

First Order Date

Total Sales

Average Sales per Order

Number of Customers

Number of Orders

Number of Products Sold

Final Conclusion

The dashboard provides a clear side-by-side comparison of two regions’ sales performance. Management can quickly identify which region is performing better, monitor sales trends, and make data-driven decisions to optimize operations and improve sales strategies.
